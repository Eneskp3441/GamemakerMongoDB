> With this function, you determine which parts of the database you will operate on.

------------

### Syntax:
```javascript
    mongo_add_data_many(dataArray);
```
------------


### Returns:
`None`

------------

### Example:
```javascript
    my_database_1.mongo_add_data_many( [ 
		{"name" : "Enes", "score" : 0 } ,
		{"name" : "Koyan", "score" : 0 } ,
		{"name" : "Roalinn", "score" : 0 } ,
]);
 ```
