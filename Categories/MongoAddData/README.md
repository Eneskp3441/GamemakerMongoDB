> With this function, you determine which parts of the database you will operate on.

------------

### Syntax:
```javascript
    MongoInit(dataSource, database, collection);
```
------------


### Returns:
`Database Object`

------------

### Example:
```javascript
    my_database_1= new MongoInit("Eneskp3441","GamemakerStudioTest","users")
 ```
